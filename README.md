# People Mobs Mod2

for Minecraft 1.14.4 (Forge 28.1)

## ↓↓ download ↓↓

**Lastest** [peoplemobsmod2-0.1.3.jar](https://github.com/dododoshirouto/PeopleMobsMod2/releases/tag/0.1.3)

## About

Mobの見た目を女の子にするMODです。

既存Mobの見た目を変えるMODです。

**Mobを増やすMODではありません**

このMODが完成したら、**かわいい女の子に囲まれた**生活ができるようになります。
